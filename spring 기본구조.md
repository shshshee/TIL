# Spring 기본구조 


## model

- 데이터 처리 부분

## view

- 프레젠테이션 로직을 담당
- 일반적으로 자바는 JSP를 통해 구현한다

## controller

- 요쳥처리 ( 뷰와 모델사이의 중간조정 역할 )
- 수행한 비즈니스 로직의 결과를 뷰를 통해 보여줌
